# Revenue Pipeline Tracker

## Pipeline Stages

### Stage 1: MQL (Marketing Qualified Lead)

Entry criteria:
- Downloaded resource (automation guide, case study)
- Attended webinar or workshop
- Requested consultation via website form
- Engaged with 3+ LinkedIn posts or emails

### Stage 2: SQL (Sales Qualified Lead)

Entry criteria:
- Discovery call scheduled or completed
- Budget confirmed (can afford Rs 50k+ implementation)
- Decision-maker identified
- Timeline confirmed (looking to start within 3 months)

### Stage 3: Proposal / Deal Stage

Entry criteria:
- Proposal sent with pricing breakdown
- Implementation scope defined
- Contract template prepared
- Payment terms confirmed

### Stage 4: Closed-Won (Paid Client)

Entry criteria:
- Contract signed
- First payment (50% implementation deposit) received
- Project kickoff scheduled
- CRM marked as "Client - Active"

## ICP-Specific Conversion Rates

### SaaS Startups (Post-Seed, 5-25 employees)

- Primary pain: Manual ops slowing scale
- MQL to SQL: 30-35%
- SQL to Proposal: 65-70%
- Proposal to Won: 40-45%
- Overall MQL to Won: 8-10%
- Typical cycle: 30-50 days

### Logistics and Supply Chain SMEs (50-200 employees)

- Primary pain: Manual tracking, inefficient coordination
- MQL to SQL: 20-25%
- SQL to Proposal: 55-60%
- Proposal to Won: 30-35%
- Overall MQL to Won: 3-5%
- Typical cycle: 60-90 days

### Fintech and Payment Processors (10-100 employees)

- Primary pain: Regulatory reporting, fraud detection automation
- MQL to SQL: 25-30%
- SQL to Proposal: 50-55%
- Proposal to Won: 35-40%
- Overall MQL to Won: 5-7%
- Typical cycle: 45-75 days

### Professional Services (HR, Legal, Consulting) (10-50 employees)

- Primary pain: Admin overhead, client communication
- MQL to SQL: 15-20%
- SQL to Proposal: 45-50%
- Proposal to Won: 25-30%
- Overall MQL to Won: 2-4%
- Typical cycle: 50-80 days

## Drop-Off Patterns and Mitigation

### Drop-Off 1: MQL to SQL (70-75% loss)

Primary causes:
- Tire-kickers (downloaded content but no real intent)
- Budget mismatch (SME cannot afford even basic package)
- No clear pain point identified
- Lost contact (email bounce, no response to outreach)

Mitigation:
- Tighten MQL criteria with lead scoring
- Add budget qualifier question in lead forms
- Implement 3-touch nurture sequence before SQL qualification

### Drop-Off 2: SQL to Proposal (30-40% loss)

Primary causes:
- Scope creep discovered in discovery (project too complex)
- Timeline mismatch (client needs immediate solution, delivery takes 4-6 weeks)
- Decision-maker not involved in discovery call
- Competitive evaluation (comparing with freelancers or other agencies)

Mitigation:
- Validate technical feasibility before SQL stage
- Include decision-maker attendance as SQL requirement
- Develop battle card comparing Layaa AI vs. freelancers vs. DIY

### Drop-Off 3: Proposal to Closed-Won (55-65% loss)

Primary causes:
- Sticker shock (implementation cost perceived as high)
- Unclear ROI calculation (client does not see value in retainer model)
- Legal/contract concerns (IP ownership, data security clauses)
- Internal prioritization shift (client postpones AI adoption)

Mitigation:
- Create ROI calculator for proposals (show payback period)
- Prepare client-friendly contract summary (1-pager)
- Add urgency trigger ("Limited implementation slots")
- Offer payment plan option

## Escalation Triggers (Auto-Alert Conditions)

1. MQL to SQL conversion drops below 15% for 2 consecutive weeks.
2. Any stage conversion rate declines by >10 percentage points month-over-month.
3. Average sales cycle exceeds 120 days for 3+ deals.
4. Closed-Won rate drops below 25% for 2 consecutive months.
5. Pipeline value falls below Rs 15L.
6. 3+ deals stuck in Proposal stage for >60 days.

Action on any trigger: Generate Revenue Bottleneck Report and escalate immediately.

---

## Attribution Model

Model: First-touch attribution (campaign that generated initial MQL).
Fallback: If multi-touch, credit the campaign with the highest engagement score.

ROI Score thresholds:
- Very High: Closed-Won Rate >30%, CAC <Rs 20k
- High: Closed-Won Rate 20-30%, CAC Rs 20-40k
- Medium: Closed-Won Rate 10-20%, CAC Rs 40-60k
- Low: Closed-Won Rate <10%, CAC >Rs 60k

---

## Reporting Cadence

**Daily:** Pipeline additions/removals logged in CRM. Stage movement tracked (automated).

**Weekly:** Pipeline Health Scorecard sent to strategic leadership. Drop-off alerts flagged to relevant owners.

**Monthly:** Full Revenue Performance Snapshot. ICP-level conversion analysis. Campaign ROI review.

**Quarterly:** Benchmark recalibration. Forecast accuracy audit. Memory update proposals for persistent bottlenecks.
