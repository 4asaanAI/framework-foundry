# Tara — KB — Content Format Templates

## LinkedIn Short-Form Post

```
[HOOK — 1-2 sentences. Provocative, personal, or counter-intuitive. Stops the scroll.]

[INSIGHT — 2-3 sentences. The core point. What you've learned, observed, or believe.]

[EVIDENCE/EXAMPLE — 2-4 sentences. Specific story, data point, or real-world example.]

[TAKEAWAY — 1-2 sentences. What the reader should think about or do differently.]

[CTA — 1 sentence. Subtle engagement prompt.]
```
**Specs:** 150-300 words. Line breaks between ideas. Conversational tone.

## LinkedIn Long-Form Post

```
[HOOK — Strong opening that frames the problem or counter-narrative]
[CONTEXT — Why this matters now. 2-3 sentences.]
[SECTION 1 — First key point with supporting evidence]
[SECTION 2 — Second key point with supporting evidence]
[SECTION 3 (Optional) — Third point or reframe]
[SYNTHESIS — Pull it together. What does this mean?]
[CTA — What should the reader do, think, or share?]
```
**Specs:** 500-800 words. Subheadings optional. Still conversational.

## Instagram Reel Script

```
[HOOK — First 3 seconds. Visual + spoken. Grab attention.]
"[Opening line — provocative or surprising]"

[CONTEXT — 5-10 seconds. Set up the problem or story.]

[INSIGHT/REVEAL — 10-15 seconds. The core point.]

[CTA — Last 3-5 seconds. What to do next.]
```
**Specs:** 50-100 words spoken. Building-in-public tone. First-person. Authentic.

## Email Campaign

```
Subject Line: [6-8 words — clear, benefit-focused, no clickbait]

[HOOK — 1 sentence. Personal or provocative opener.]
[VALUE — 2-3 sentences. What you are offering and why it matters.]
[CTA — 1 sentence. Clear, single action.]
[SIGN-OFF — Founder name if attributed, or "Team Layaa AI"]
```
**Specs:** 100-200 words. Short paragraphs. Bold CTA. No excessive formatting.

## Blog Post

```
# [Title — Clear, benefit-focused, SEO-aware]

## Introduction [150-200 words — Hook, problem, preview solution]
## [Section 1] [200-300 words — Problem deep-dive or context]
## [Section 2] [200-300 words — Solution or framework]
## [Section 3] [200-300 words — Evidence, case study, example]
## [Section 4 — Optional] [200-300 words]
## Conclusion [100-150 words — Summarize, restate value, CTA]
```
**Specs:** 1,000-2,000 words. Subheadings every 200-300 words. Educational CTA.

## Whitepaper

```
# [Title]
## Executive Summary [250-500 words]
## 1. Problem Analysis [500-800 words]
## 2. Market Context [300-500 words]
## 3. Methodology / Framework [500-800 words]
## 4. Solution Deep Dive [800-1,200 words]
## 5. Case Evidence / Results [300-500 words]
## 6. Recommendations [300-500 words]
## References
```
**Specs:** 2,500-5,000 words. All data requires [EVIDENCE: VALIDATED]. Formal but accessible.
