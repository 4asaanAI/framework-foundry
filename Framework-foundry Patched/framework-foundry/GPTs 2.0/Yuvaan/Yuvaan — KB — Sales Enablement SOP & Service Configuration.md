## Sales Enablement SOP

### New Deal Entry
1. Prospect identified (inbound inquiry or outbound target)
2. Run `/account-research` to evaluate ICP fit
3. Create deal record in pipeline with stage: Raw Lead
4. If ICP fit confirmed → move to MQL, schedule discovery
5. Run `/call-prep` before discovery call
6. After discovery → run `/call-summary` to extract action items
7. If qualified → move to SQL, begin proposal preparation

### Proposal Preparation
1. Gather discovery data from Arjun/Yuvaan's call notes
2. Request pricing from @Veer using `/forecast` or direct @mention
3. Request brand review from @Tara for client-facing language
4. Generate proposal using `/proposal-generator`
5. Review against Quality Checklist
6. Submit to Founders for approval before sending
7. Once approved → deliver to prospect and update pipeline stage

### Post-Call Follow-Up
1. Run `/call-summary` within 2 hours of call
2. Draft follow-up email with key discussion points and next steps
3. Update deal record with call outcomes, objections raised, next steps
4. Create tasks for any promised deliverables (proposals, info requests)
5. Schedule next touchpoint
