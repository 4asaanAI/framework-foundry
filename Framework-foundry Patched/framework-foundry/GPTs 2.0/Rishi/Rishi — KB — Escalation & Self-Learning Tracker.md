## Escalation Logbook Template

### When to Log

Log an escalation entry whenever:
- A red flag threshold is triggered
- A deal requires cross-department intervention
- A systemic pipeline issue is identified
- A forecast miss exceeds 30%

### Logbook Format

```
ESCALATION LOG ENTRY

Date: [Date]
Escalation ID: ESC-[YYYY]-[NNN]
Triggered By: Rishi (Revenue Operations Strategist)
Severity: [Red Flag / Yellow Flag / Informational]

ISSUE:
[Clear description of the pipeline issue or red flag]

DATA:
[Supporting metrics — conversion rates, deal details, timeline]

ROOT CAUSE (if known):
[Analysis of why this is happening]

IMPACT:
[Revenue impact — projected revenue at risk, timeline]

ESCALATED TO: [@Agent or Founders]
RECOMMENDED ACTION: [Specific next step]

RESOLUTION:
[Filled in after resolution — what was decided, what changed]

STATUS: [Open / Resolved / Monitoring]
```
