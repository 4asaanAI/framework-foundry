## 1. Red Flags to Monitor

| Signal | Threshold | Action |
|--------|-----------|--------|
| CAC rising | >20% increase over 2 weeks | Pause lowest-performing channels, audit targeting |
| MQL quality dropping | SQL rate <15% | Review targeting, tighten qualifying criteria |
| Funnel drop-off spike | >30% drop at any single stage | Investigate stage-specific issues, A/B test fixes |
| Channel CPA above 2x target | Sustained for 7+ days | Pause channel, review strategy |
| Negative ROAS | Any channel for 14+ days | Kill the campaign, reallocate budget |
| Email unsubscribe spike | >2% on any send | Review content, frequency, and list quality |
| Landing page bounce | >70% | Review page speed, message match, UX |
| Ad fatigue | CTR drops >30% from initial | Refresh creative, rotate audiences |

---

## 2. Performance Review Cadence

- **Daily:** Quick check on active paid campaigns (spend pacing, CTR, CPA)
- **Weekly:** Channel performance summary, funnel metrics, budget pacing
- **Monthly:** Full performance audit, channel comparison, strategy adjustments
- **Quarterly:** Growth strategy review with Mira, GTM alignment check

## 4. Funnel Conversion Targets

| Stage | Target Rate | Red Flag | Measurement |
|-------|-------------|----------|-------------|
| Visit -> MQL | 2-5% | <1% | Form submissions / total visits |
| MQL -> SQL | 25% | <15% | Qualified by sales criteria |
| SQL -> Proposal | 60% | <40% | Proposals sent / SQLs |
| Proposal -> Won | 35% | <25% | Deals closed / proposals sent |
| End-to-End | 5-8% | <3% | MQL to customer |
| Sales Cycle | 42-65 days | >120 days | First touch to close |
