## Financial Close Checklist

### Monthly Close (By 5th of following month)

- [ ] All invoices for the month issued
- [ ] All expenses recorded and categorized
- [ ] Bank reconciliation completed
- [ ] GST data prepared and shared with CA
- [ ] TDS computed and deposited (by 7th)
- [ ] Cash flow statement updated
- [ ] P&L statement prepared
- [ ] Infrastructure budget check (Rs.500 cap)
- [ ] Outstanding receivables reviewed and aging updated
- [ ] Vendor payments processed as per terms
- [ ] Any discrepancies investigated and resolved

### Quarterly Close (Additional)

- [ ] TDS return data prepared and shared with CA
- [ ] Quarterly P&L and cash flow summary for Founders
- [ ] Budget vs. actual variance analysis
- [ ] Update financial projections for next quarter
- [ ] Review vendor contracts for renewal/renegotiation
- [ ] Advance tax assessment (if applicable)

### Annual Close (FY End — March 31)

- [ ] All monthly closes completed for the year
- [ ] Full-year P&L, Balance Sheet, Cash Flow prepared
- [ ] Year-end bank reconciliation (final)
- [ ] Depreciation schedules updated (if assets exist)
- [ ] Tax computation prepared for CA
- [ ] Audit preparation — organize all supporting documents for AOC-4
- [ ] Coordinate with Anne on annual filing timeline
- [ ] TDS annual return preparation
- [ ] GST annual return preparation (GSTR-9)
- [ ] Coordinate financial data for AGM documents
