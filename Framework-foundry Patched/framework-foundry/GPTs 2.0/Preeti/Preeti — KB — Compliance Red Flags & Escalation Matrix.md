# Preeti — KB — Compliance Red Flags & Escalation Matrix

> Red flag identification, escalation paths, risk register, and compliance scorecard framework.

---

## Immediate Escalation (Critical)

| Red Flag | Action |
|----------|--------|
| Personal data processed without consent mechanism | Stop processing, escalate to Founders |
| Children's data collected without parental consent (EduFlow) | Stop collection, escalate immediately |
| Data breach detected or suspected | Trigger incident response, notify Founders + Kaiser |
| Cross-border data transfer without assessment | Block transfer, assess compliance |
| Client requests data deletion and no mechanism exists | Escalate, implement mechanism |
| Regulatory inquiry or notice received | Escalate to Founders + Abhay immediately |

---

## High Priority (Address Within 7 Days)

| Red Flag | Action |
|----------|--------|
| Privacy policy does not cover a new data processing activity | Update policy, coordinate with Abhay |
| No data processing agreement with a client where Layaa AI processes their customer data | Flag to Abhay for DPA drafting |
| Security measure gaps identified (e.g., unencrypted data at rest) | Flag to Kaiser for remediation |
| Employee/agent accessing data beyond their role scope | Review access controls, restrict |
| Retention period exceeded for stored personal data | Initiate data deletion process |

---

## Medium Priority (Address Within 30 Days)

| Red Flag | Action |
|----------|--------|
| Compliance documentation not updated for >90 days | Schedule review and update |
| New regulation announced with potential impact | Assess impact, document, brief Kabir |
| Client in regulated sector without compliance assessment | Conduct assessment before engagement proceeds |
| Internal audit checklist items incomplete | Complete or schedule completion |

**Scoring Guide:**
- 9-10: Exceeds requirements — best practice compliance
- 7-8: Meets requirements — compliant with minor improvements needed
- 5-6: Partially compliant — gaps exist that need attention within 30 days
- 3-4: Significantly non-compliant — gaps require immediate attention
