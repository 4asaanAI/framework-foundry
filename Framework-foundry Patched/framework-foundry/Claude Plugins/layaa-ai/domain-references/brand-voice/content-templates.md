# Layaa AI — Content Format Templates

## LinkedIn Short-Form (Thought Leadership)
**Length:** 150-300 words | **Reading time:** 30-60 seconds
```
HOOK (1-2 sentences): Observation, question, or contrarian statement
INSIGHT (2-4 sentences): Core idea grounded in experience
CONTEXT (2-3 sentences): Brief example or data point
TAKEAWAY (1-2 sentences): Why this matters
CTA (optional): "What's your take?" / "Curious what others think"
```
**Example:** "Most AI tools fail because they solve the wrong problem. They focus on what AI can do. Not what teams actually need done..."

## LinkedIn Long-Form
**Length:** 500-800 words | **Reading time:** 2-3 minutes
```
HOOK (2-3 sentences): Strong statement or story
PROBLEM (3-5 sentences): Frame the challenge
INSIGHT 1 (4-6 sentences): First point + example
INSIGHT 2 (4-6 sentences): Second point + example
INSIGHT 3 (optional): Third point if needed
SYNTHESIS (3-4 sentences): Connect back to bigger picture
TAKEAWAY (2-3 sentences): Concrete conclusion
```

## Instagram Reel Scripts (Building in Public)
**Length:** 30-60 seconds | **Structure:**
```
HOOK (3-5 seconds): Stop-scroll statement
CONTEXT (5-10 seconds): What we're doing at Layaa AI
INSIGHT (10-20 seconds): What we learned / what happened
PAYOFF (5-10 seconds): Why it matters
CTA (3-5 seconds): Follow for more / link in bio
```
**Tone:** Authentic, behind-the-scenes, casual but insightful

## Website Landing Page Copy
```
HERO: Headline (5-10 words, benefit-first) + Subhead (1 sentence) + CTA button
PROBLEM: 2-3 pain points the visitor relates to
SOLUTION: How Layaa AI solves it (3 bullet points max)
SOCIAL PROOF: Results, logos, or testimonials (if validated)
HOW IT WORKS: 3-step process
CTA: Repeat with urgency or specificity
```

## Email Campaign Sequences
**Per email:** 100-200 words
```
SUBJECT LINE: 6-8 words, curiosity or benefit-driven
HOOK: 1 sentence that earns the next sentence
VALUE: 2-3 sentences delivering insight or offer
CTA: 1 sentence, single clear action
```
**Sequence structure:** Welcome → Value → Case Study → Offer → Follow-up

## Blog Posts
**Length:** 1000-2500 words
```
INTRO: Hook + context + what reader will learn
PROBLEM: What's broken / what most people get wrong
SOLUTION: Layaa AI's approach / framework / methodology
EVIDENCE: Examples, data, case references
CONCLUSION: Summary + next steps
```
**Subheadings** every 200-300 words. Bullets for key points.

## Campaign Launch Messaging
```
ANNOUNCEMENT: What's new + why it matters (2-3 sentences)
VALUE PROP: How this helps the target audience (2-3 bullets)
PROOF: Early results, beta feedback, or logical argument
CTA: Clear next step with deadline if applicable
```

## CTA Frameworks
| Context | CTA Style | Examples |
|---------|-----------|---------|
| Awareness | Soft, educational | "Learn more", "See how it works" |
| Consideration | Value-driven | "Get a free assessment", "See pricing" |
| Decision | Action-driven | "Book a demo", "Start your pilot" |
| Retention | Engagement | "Share your results", "Refer a colleague" |
