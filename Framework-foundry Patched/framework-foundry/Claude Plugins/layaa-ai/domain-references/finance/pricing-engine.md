# Layaa AI — Pricing Engine

## Hybrid Pricing Formula

```
Total Price = Fixed Base + Variable Component + Value-Based Premium
```

- **Fixed Base:** Minimum project fee by service vertical
- **Variable Component:** Complexity × Estimated Hours × Hourly Rate
- **Value-Based Premium:** Applied when measurable client ROI justifies premium pricing (e.g., cost savings >10x fee)

## Complexity Multipliers

| Level | Multiplier | Characteristics |
|-------|-----------|----------------|
| Simple | 1.0x | Single tool, <3 integrations, standard workflow, no custom logic |
| Moderate | 1.5x | 2–3 tools, 3–5 integrations, custom logic, some API work |

## Package Tiers & Deal Sizing

### Starter Package
- **Implementation:** ₹50k–₹1.5L
- **Monthly Retainer:** ₹15k/month
- **Includes:** Basic automation (1–2 workflows), email support, monthly review
- **Best for:** First-time AI adopters, single-process automation, proof of concept

### Growth Package
- **Implementation:** ₹1.5L–₹5L
- **Monthly Retainer:** ₹40k/month
- **Includes:** Multi-workflow automation (3–5 workflows), priority support, bi-weekly reviews, minor enhancements
- **Best for:** Scaling operations, multi-department automation, established SMEs

### Enterprise Package
- **Implementation:** ₹5L+
- **Monthly Retainer:** Custom (typically ₹80k–₹1.2L/month)
- **Includes:** Full-stack automation, dedicated support, weekly reviews, SLA guarantees, custom reporting
- **Best for:** Complex multi-system integrations, regulated industries, high-volume operations

## Pricing Rules

| Rule | Detail |
|------|--------|
| Minimum project value | ₹50,000 |
| Upfront deposit | 50% of implementation fee on contract signing |
| Retainer billing | Monthly in advance, net-15 payment terms |
| Late payment interest | 1.5% per month |
| Service suspension | Right to suspend after 30 days overdue |
| Payment milestones (large projects) | 50% / 25% / 25% (signing / development complete / go-live) |

---

## Discount Authority

| Discount Range | Approval Required |
|---------------|-------------------|
| Up to 10% | Sales can offer without approval |
| 10–20% | Veer (Pricing Specialist) approval |
| >20% | Founder approval only |
| Custom pricing outside standard tiers | Founder approval only |

**Discount justification required for all discounts >10%:** Must document reason (competitive pressure, strategic account, multi-year commitment, etc.)

---

## Red Flag Thresholds

| Metric | Threshold | Action |
|--------|-----------|--------|
| Gross margin below 40% | Escalate to Founders — do not proceed without approval |
| Gross margin below 30% | Reject deal unless Founder-approved strategic exception |
| Implementation fee below ₹50k | Reject — below minimum viable project |
| Retainer below ₹15k/month | Only for Starter tier with limited scope |
| Scope exceeding estimate by >30% | Re-scope and reprice before proceeding |
| Client requesting >20% discount | Founder approval with documented justification |
