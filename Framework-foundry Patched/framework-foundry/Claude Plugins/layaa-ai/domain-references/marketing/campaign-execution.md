# Layaa AI — Campaign Execution Framework

## Core Execution Principles
**NEVER START WITHOUT:** Complete campaign brief, approved messaging/creative, budget confirmation, clear success metrics
**ALWAYS INCLUDE:** Tracking tags, setup documentation, quality check, performance monitoring plan
**ESCALATE IMMEDIATELY IF:** Brief incomplete, budget unclear/exceeded, technical issues, strategy-execution mismatch

## Campaign Execution Workflow

### Step 1: Intake & Validation
- Campaign brief complete (objective, audience, channels, CTAs, metrics, timeline)
- Strategic approval from Mira or Zoya
- Creative assets ready from Tara
- Budget allocated (Anne confirms if above threshold)
- No conflicts with other active campaigns
- **If ANY item missing → ESCALATE to Mira, do not proceed**

### Step 2: Setup & Configuration
- Create campaign in platform (email, CRM, ad platform)
- Set up audience segments based on targeting criteria
- Configure automation triggers (for drip campaigns)
- Add UTM tracking tags (standard format below)
- Test on sample list/audience

### UTM Tag Standard
```
utm_source=[channel] (linkedin, google, meta, email)
utm_medium=[type] (paid, organic, email, referral)
utm_campaign=[campaign_name] (lowercase, underscores)
utm_content=[variant] (a_b_test_variant)
utm_term=[audience_segment]
```
Rules: Always lowercase, underscores not spaces, naming convention: [audience]_[topic]_[month][year]

### Step 3: Quality Check (Before Launch)
- All tracking verified and working
- Landing pages live, forms functional, mobile-responsive, <3s load time
- CTA routing confirmed (where leads go after conversion)
- A/B tests properly configured
- Budget caps set correctly
- **Quality Gate: Do not launch until ALL items checked**

### Step 4: Launch & Monitor
- Launch at optimal time for target audience
- Monitor first 2-4 hours for technical issues
- Check delivery rates, bounce rates, error rates
- Verify tracking data flowing correctly
- Document any launch issues

### Step 5: Optimization
- Daily: Monitor spend, CPL, conversion rates
- Weekly: Analyze performance vs. benchmarks, adjust bids/targeting
- Bi-weekly: Creative refresh if fatiguing (CTR declining >20%)
- Monthly: Full performance review with Mira/Zoya

### Step 6: Reporting & Handoff
- Weekly performance summary to Mira
- Monthly campaign retrospective
- Lead handoff to sales (routing, scoring, assignment)
- Archive campaign data and learnings

## Campaign Brief Template (Quick Reference)
1. Campaign name and type (awareness/consideration/conversion/retention)
2. Objective + primary KPI + target value
3. Target persona (from ICP profiles)
4. Channel strategy + budget allocation
5. Messaging + creative requirements
6. Timeline (launch → end date)
7. Success criteria + reporting cadence
