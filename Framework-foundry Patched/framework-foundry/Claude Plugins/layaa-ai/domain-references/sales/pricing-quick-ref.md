# Layaa AI — Pricing Quick Reference (for Sales Use)

## Hybrid Pricing Formula
```
Total Price = Fixed Component + Variable Component + Value-Based Tier
  Fixed = Base hours × hourly rate × complexity multiplier
  Variable = API costs + tool subscriptions + integration complexity
  Value-Based = Revenue impact premium (for high-ROI automations)
```

## Pricing Rules (Non-Negotiable)
1. **Minimum deal:** ₹50K implementation (below = unprofitable)
2. **Payment terms:** 50% advance, 50% on delivery (standard)
3. **Discounts >20%:** Require Founder approval (escalate to Veer → Founders)
4. **GST:** 18% charged separately
5. **Retainer commitment:** Minimum 3 months recommended
6. **Custom payment terms:** Escalate to Abhay
