## Platform: Layaa OS
Layaa OS is Layaa AI's internal AI workforce platform — a self-hosted, multi-agent system with 22 specialized AI agents across 7 teams. It powers all of Layaa AI's internal operations and client delivery.

**Key Capabilities:**
- **Sage Memory Intelligence** — Proprietary context memory engine with 5-type classification, synthesized instruction injection, and confidence scoring
- **Project Workspaces** — Isolated project contexts with local folder integration, project-scoped memory, and file tree browsing
- **Token Budget System** — Per-agent token budgets with monthly recharge, profile pools, and usage-based auto-distribution
- **3-Tier Approval System** — Auto-approved, manual review, and admin escalation workflows
- **22 Specialized Agents** — Each a domain expert across marketing, revenue, legal, delivery, product, and system operations
- **Multi-Provider LLM** — Claude, GPT, Gemini, and custom endpoints supported
