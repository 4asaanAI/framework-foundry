## What We Do
AI automation and enablement firm empowering Indian SMEs to modernise operations through applied AI. We assess client operations, identify automation opportunities, build custom AI workflows using no-code/low-code tools, and train teams to use them. Clients typically see ROI within 30-60 days of deployment.

## Delivery Methodology
**Discovery → Assessment → Development → Validation → Enablement**

1. **Discovery** — Business context analysis, current state mapping, feasibility validation, risk identification → Validated scope doc
2. **Assessment** — Process mapping, automation opportunity scoring, technology recommendation, ROI projection → AI Adoption Roadmap
3. **Development** — Architecture design, 7-stage workflow building, iterative builds → Working automation system
4. **Validation** — End-to-end testing, edge case testing, security/compliance verification → Test report + sign-off
5. **Enablement** — User training, admin training, runbooks, go-live support → Trained team + documentation

## 4 Ideal Customer Profiles

### ICP 1: SaaS Startups
- **Size:** 5-25 employees, post-seed | **Cycle:** 30-50 days | **Budget:** ₹2-5L implementation
- **Pain:** Manual ops slowing scale — founders doing everything
- **Personas:** Founder/CEO, Head of Ops
- **Message:** "Scale without hiring. Automate the repetitive so you can focus on product."
- **Objections:** "We'll build it in-house later" / "Is this enterprise-grade?"

### ICP 2: Logistics & Supply Chain SMEs
- **Size:** 50-200 employees | **Cycle:** 60-90 days | **Budget:** ROI-driven
- **Pain:** Manual tracking, inefficient coordination, paper-based workflows
- **Personas:** Operations Manager, Business Owner, Fleet/Warehouse Manager
- **Message:** "Replace spreadsheets and phone calls with automated coordination."
- **Objections:** "Our processes are too unique" / "Team won't adopt new tools"

### ICP 3: Fintech & Payment Processors
- **Size:** 10-100 employees | **Cycle:** 45-75 days | **Budget:** Medium-High
- **Pain:** Regulatory reporting burden, fraud detection gaps, manual compliance
- **Personas:** CTO, Compliance Head, Product Manager
- **Message:** "Automate compliance and reporting so your team builds product, not paperwork."
- **Objections:** "Security concerns" / "Regulatory requirements make automation risky"

### ICP 4: Professional Services (HR/Legal/Consulting)
- **Size:** 10-50 employees | **Cycle:** 50-80 days | **Budget:** Variable
- **Pain:** Admin overhead, client communication gaps, document-heavy workflows
- **Personas:** Managing Partner, Practice Head, Office Manager
- **Message:** "Automate admin so your experts spend time on billable work."
- **Objections:** "Our work is too nuanced for AI" / "Clients expect personal touch"
