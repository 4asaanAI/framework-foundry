## Sub-Skill Execution

### campaign-plan
1. Define: objective (awareness/leads/conversion), target segment, timeline, budget
2. Select ICP and craft persona-specific message angle
3. Choose channels based on priority order and objective
4. Build campaign brief: objective → audience → key message → channels → creative → timeline → KPIs
5. For Layaa AI: apply messaging pillars, brand voice, Indian business context

### draft-content
1. Identify: content type, audience, key message, CTA
2. Apply brand voice: confident, practical, outcome-led, India-specific
3. Draft with: hook (pain/insight) → value (what changes) → proof (numbers/example) → CTA
4. For founder content: ask which founder — Abhimanyu (strategic/visionary) or Shubham (technical/pragmatic)
5. Provide 2-3 headline/hook variants

### content-creation
Full multi-channel package for a given topic:
1. Blog post (500-800 words, SEO-optimised)
2. LinkedIn company post
3. LinkedIn founder post (personal angle)
4. Email newsletter section (200 words)
5. Short social caption
Apply consistent messaging angle across all formats.

### email-sequence
1. Define: sequence goal, audience segment, number of emails
2. Map timing: Day 0 → Day 3 → Day 7 → Day 14 (adjust based on goal)
3. Write each email: subject line, preview text, body (under 200 words), CTA
4. Add branching: what happens if they click? If they don't open?
5. Apply Layaa brand voice throughout

### social-media-calendar
1. Ask for: week dates, events/announcements, preferred themes
2. Build: Company LinkedIn (4 posts) + Founder LinkedIn (3 posts — specify which founder or both)
3. For each post: type (insight/case study/tip/question/announcement), hook, body, CTA, hashtags
4. Vary formats: text-only, carousel idea, image suggestion, video idea
5. Note optimal posting times and which posts to boost with paid

### performance-report
1. Ask for: impressions, engagement, clicks, leads, MQLs, spend (if paid)
2. Compare to benchmarks: LinkedIn organic CTR 0.5-1%, email open rate 20-30%, lead form CVR 2-5%
3. Identify: top performing content, underperforming channels, trend direction
4. Build: metrics → trends → wins → misses → next period recommendations

### seo-audit
1. Define: target keywords, competitor URLs (if known), current rankings (if known)
2. Research: search volume, difficulty, intent for target keywords
3. On-page audit: title tags, meta descriptions, content gaps, internal linking
4. Identify 3-5 quick wins (low difficulty, high relevance)
5. Recommend: content to create, pages to optimise, backlink opportunities

### competitive-brief
1. Research specified competitor: positioning, messaging, target audience, pricing (if public)
2. Compare to Layaa AI: where do we win? Where are we weaker?
3. Identify messaging gaps competitor hasn't claimed that Layaa AI can own
4. Build: competitor overview → comparison table → Layaa AI differentiation → recommended response

### brand-review
1. Receive content to review
2. Check against: voice attributes, DO/DON'T list, vocabulary guidance (from Project Knowledge)
3. Flag: hype language, passive voice, missing Indian context, missing numbers, vague claims
4. Score: 1-10 brand alignment
5. Rewrite flagged sections with tracked changes explained
