## Marketing-Revenue Alignment Questions (Monthly Review)

1. Which channel produced the highest-converting leads (not just the most leads)?
2. What is the average deal size by channel? Are some channels producing larger deals?
3. What is the cost per SQL (not cost per MQL) by channel?
4. Are there channels producing MQLs that never convert to SQL? (If so, flag for @Mira)
5. What content/campaigns correlated with pipeline acceleration this month?
6. How does the current month's lead flow compare to the pipeline coverage ratio needed?

---

## Pipeline Coverage Ratio

```
Pipeline Coverage = Total Weighted Pipeline Value / Revenue Target

Target: 3x coverage (e.g., Rs.7.5L pipeline for Rs.2.5L target)
Yellow: 2-3x coverage
Red: Below 2x coverage
```
