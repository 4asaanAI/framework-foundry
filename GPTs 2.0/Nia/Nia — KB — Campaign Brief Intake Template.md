## Campaign Brief Intake Template

Use this template to document and confirm campaign briefs received from Mira:

```
CAMPAIGN BRIEF INTAKE
Received From: [Mira / Kabir / Founders]
Date Received: [Date]
Campaign Name: [Name]
Intake Status: [Complete / Incomplete — see gaps below]

STRATEGIC ELEMENTS (From Mira):
- [ ] Campaign Objective: [Specific, measurable goal]
- [ ] Target Segment: [ICP, firmographics, decision maker]
- [ ] Key Messages: [Primary and supporting messages]
- [ ] Content Pillars: [P1-P5 reference]
- [ ] Success Metrics: [KPIs and targets]
- [ ] Timeline: [Start date, end date, key milestones]
- [ ] Budget: [Total amount, approved by whom]

CHANNEL ELEMENTS (From Zoya):
- [ ] Channel Mix: [Primary, secondary, supporting]
- [ ] Targeting Parameters: [Platform-specific targeting]
- [ ] Creative Specifications: [Format, size, quantity]
- [ ] Performance Benchmarks: [Expected CTR, CPA, conversion]
- [ ] Budget Allocation: [By channel]

CONTENT ELEMENTS (From Tara):
- [ ] Ad Copy: [All variants]
- [ ] Email Copy: [All emails in sequence]
- [ ] Landing Page Copy: [Full page content]
- [ ] Social Media Posts: [All scheduled posts]
- [ ] Visual Assets: [Images, videos, graphics]

SALES ELEMENTS (From Yuvaan):
- [ ] CTA Language: [Approved CTAs]
- [ ] Lead Scoring Rules: [Criteria and thresholds]
- [ ] Handoff Triggers: [When to pass to sales]
- [ ] Follow-Up Sequence: [Post-handoff process]

GAPS IDENTIFIED:
1. [Gap 1 — requested from whom, by when]
2. [Gap 2 — requested from whom, by when]
3. [Gap 3 — requested from whom, by when]

EXECUTION PLAN:
- Setup Start: [Date]
- Testing Complete: [Date]
- Launch Date: [Date]
- First Review: [Date]
- Mid-Campaign Review: [Date]
- Campaign End: [Date]
- Post-Campaign Report: [Date]
```
