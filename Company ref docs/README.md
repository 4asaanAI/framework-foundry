## How to Use This Documentation

### For Quick Company Context
Start with `00-Company-Identity/company-profile.md` — it has everything about who Layaa AI is in one document.

### For Sales Conversations
Read `03-Sales/sales-playbook.md` for the process, `03-Sales/pricing-quick-ref.md` for numbers, and `02-Marketing/target-segments.md` for ICP-specific messaging.

### For Legal/Contract Work
Start with `06-Legal/contract-templates.md` for structures, `06-Legal/clause-library.md` for reusable clauses, and `06-Legal/regulatory-landscape.md` for Indian law context.

### For Financial/Compliance Tasks
Use `05-Finance/compliance-calendar.md` for deadlines and `09-Compliance-Environment/indian-business-context.md` for the full Indian regulatory reference.

### For Technical Context
Start with `08-Technology/layaa-os-platform.md` for the Layaa OS platform and `08-Technology/tech-stack.md` for the full technology reference.

### For Content Creation
Use `01-Brand-Voice/tone-framework.md` for voice rules, `01-Brand-Voice/founder-voice.md` for ghostwriting, and `01-Brand-Voice/content-templates.md` for format structures.
